Español:

Botones Generales:

 - Esc -> salir de la pantalla actual o del programa.
 - F_12 -> Tomar captura de pantalla.

Acciones Generales:

 - Ctrl + Click -> En los elementos de la lista de descargas para seleccionar varios elementos.
 - Es posible agregar el link de la página del archivo si es de MediaFire, esto permitira al programa reanudar la descarga a pesar de que el link de descarga haya expirado.
 - Es posible instalar una extension del programa, ver en la pantalla de extras (Boton signo de interrogacion en la esquina superior de la pantalla).
 - Es posible borrar todas las descargas desde la pantalla de extras, esto borrara tanto la cache de las descargas como la base de datos.

Limitantes:

 - Existen varios links que aún no funcionan con la application, recomendable desactivar mediante la ventana de la extencion en caso de tenerla activada.
 - Problemas para que brave instale la extension, algún día subiré la extension a la web store.

English:

General Buttons:

 - Esc -> exit the current screen or the program.
 - F12 -> take a screenshot.

General Actions:

 - Ctrl + Click -> On the downloads list elements to select multiple items.
 - It is possible to add the page link of the file if it is from MediaFire, this will allow the program to resume the download even if the download link has expired.

Limitations:

 - There are several links that still do not work with the application, it is recommended to disable it through the extension window if it is activated.
